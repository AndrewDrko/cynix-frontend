🎬 **CYNIX Frontend**

Interfaz web de una aplicación de cine desarrollada con React.
Permite a los usuarios explorar películas, ver funciones, ver cines, simular comprar boletos de cine, registrarse y autenticarse.

DEMO: https://cynix-frontend-lime.vercel.app/

Tecnologías utilizadas:
- React
- Vite
- JavaScript (ES6+)
- CSS Modules
- Redux Toolkit
- React Router
- Fetch API

Características principales: 
- Visualización de películas en tiempo real
- Navegación dinámica entre vistas
- Sistema de autenticación (login / registro)
- Consumo de API REST
- Renderizado rápido con Vite
- Estilos modulares y escalables
